# Siga o Dinheiro - Conceitos de Design

## Resposta 1: Dark Historical Archive (Probabilidade: 0.08)
**Design Movement:** Arquivo Histórico Moderno com Elementos Noir

**Core Principles:**
- Fundo escuro (charcoal/preto) evocando documentos históricos
- Acentos em ouro/dourado representando o saque de recursos
- Tipografia serif para textos históricos, sans-serif para dados
- Estrutura de "livro aberto" para apresentar informações

**Color Philosophy:**
- Paleta: Preto (#1a1a1a), Ouro (#d4af37), Vermelho (#c41e3a para gastos/perdas), Cinza (#4a4a4a)
- Intenção emocional: Seriedade, autoridade, peso histórico

**Layout Paradigm:**
- Duas colunas assimétricas (receitas vs despesas)
- Divisão visual clara entre "Saque" (esquerda/ouro) e "Gastos" (direita/vermelho)
- Elementos fluem de cima para baixo em timeline vertical

**Signature Elements:**
- Ícones de moedas/ouro animados
- Linhas conectoras entre países e Alemanha
- Badges de data/evento com efeito de "carimbo histórico"

**Interaction Philosophy:**
- Hover revela detalhes adicionais
- Cliques expandem seções
- Animações suaves representando fluxo de recursos

**Animation:**
- Transições suaves (easing: cubic-bezier)
- Números contadores animados para valores financeiros
- Linhas de conexão que se desenham ao scroll

**Typography System:**
- Display: Playfair Display (serif elegante) para títulos
- Body: Inter (sans-serif limpo) para dados
- Destaque: Números em ouro com weight 700

## Resposta 2: Interactive Data Visualization (Probabilidade: 0.07)
**Design Movement:** Infografia Moderna + Dataviz Contemporânea

**Core Principles:**
- Foco em visualização de dados clara e acessível
- Cores categóricas para diferentes tipos de transações
- Gráficos interativos (Sankey, Treemap, Timeline)
- Responsivo e mobile-first

**Color Philosophy:**
- Paleta: Azul (#0066cc para receitas), Vermelho (#e74c3c para gastos), Neutro (#f5f5f5)
- Intenção: Clareza, comparação, impacto visual

**Layout Paradigm:**
- Grid de 12 colunas com seções modulares
- Gráficos ocupam espaço central
- Sidebar com filtros e legenda

**Signature Elements:**
- Gráficos Sankey mostrando fluxo de dinheiro
- Cards com estatísticas-chave
- Mini mapas dos países

**Interaction Philosophy:**
- Filtros por país/período
- Tooltips informativos
- Zoom em gráficos

**Animation:**
- Transições de dados suaves
- Animações de entrada em cascata
- Efeitos de hover nos elementos

**Typography System:**
- Roboto (sans-serif) para consistência
- Tamanhos escalonados para hierarquia
- Números em monospace para dados

## Resposta 3: Immersive Historical Experience (Probabilidade: 0.06)
**Design Movement:** Experiência Imersiva + Storytelling Visual

**Core Principles:**
- Narrativa visual que guia o usuário através da história
- Imagens de fundo impactantes (mapas, documentos históricos)
- Transições cinematográficas entre seções
- Foco em emocional + educacional

**Color Philosophy:**
- Paleta: Tons quentes (ouro, laranja) para saque, tons frios (azul, cinza) para contexto
- Intenção: Engajamento emocional, impacto histórico

**Layout Paradigm:**
- Fullscreen sections com scroll vertical
- Imagens de fundo com overlay de conteúdo
- Efeito parallax em elementos

**Signature Elements:**
- Imagens de arquivo histórico como backgrounds
- Timeline interativa com "chapters"
- Citações históricas destacadas

**Interaction Philosophy:**
- Scroll revela conteúdo progressivamente
- Cliques em eventos expandem narrativa
- Efeitos de "desvelamento" de informações

**Animation:**
- Parallax ao scroll
- Fade-in em cascata
- Transições de página cinematográficas

**Typography System:**
- Georgia (serif clássico) para narrativa
- Montserrat (sans-serif moderno) para dados
- Tamanhos grandes para impacto

## Design Escolhido: Dark Historical Archive

Escolhi a **Resposta 1 (Dark Historical Archive)** porque:
- Respeita a natureza séria e educativa do conteúdo
- A paleta ouro/preto/vermelho é visualmente impactante e significativa
- A estrutura de "livro aberto" permite apresentar receitas vs despesas de forma clara
- Elementos históricos (selos, documentos) reforçam o contexto educativo
- Funciona bem em mobile e desktop
- Evita clichês de design (sem purpura, sem Inter em tudo, sem rounded corners excessivos)

**Implementação:**
- Fundo: Preto (#0a0a0a) com textura sutil
- Acentos: Ouro (#d4af37) para receitas/saque
- Alerta: Vermelho (#c41e3a) para gastos/déficit
- Tipografia: Playfair Display + Inter
- Layout: Duas colunas assimétricas com timeline central
